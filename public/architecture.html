11 factory methods. Assembles GameEntities with components (like Unity prefabs).

ai/EnemyBrainSystem.ts
Per-frame AI for all enemies. Melee/ranged/crawler trees. A* chase, dodge-able attacks.

All Components (15)

AnimatedSpriteComponent
8-directional walk + attack animations. Spritesheet extraction. Damage frame callback. State machine: idle→walk→attack.
  • onPreUpdate: YES
  • Walk: loop, 10fps
  • Attack: one-shot, 12fps
HealthComponent
HP tracking with armor reduction, invincibility frames, death event.
  • onPreUpdate: YES (inv timer)
  • damage(amt), heal(amt)
  • alive, hp, maxHp
MeleeAttackComponent
Melee attack with cooldown, arc check, damage frame sync.
  • onPreUpdate: YES
  • canAttack, startAttack()
  • damage, range, cooldownMs
RangedAttackComponent
Fires projectile entities with trails, glow, impact effects.
  • onPreUpdate: YES
  • fire(target, scene)
  • Arrow or magic projectile
AIBrainComponent
Enemy state data: type, speed, damage, ranged flag, aggro, wander timer.
  • onPreUpdate: NO
  • Data container only
  • Used by EnemyBrainSystem
BuildingComponent
Per-type building behavior. TURRET auto-attacks nearest enemy.
  • onPreUpdate: YES (turret)
  • Fires arrow projectiles
  • 180px range, 8 dmg
ShadowCasterComponent
Draws entity sprite as black silhouette, stretched away from light.
  • onPreUpdate: YES
  • Uses graphics.onPreDraw
  • graphic.tint = Black
GridOccupancyComponent
Marks/unmarks tiles in walkability grid. Auto-cleanup on remove.
  • onPreUpdate: NO
  • onAdd → setBlocked
  • onRemove → setWalkable
LightSourceComponent
Fog-of-war light source metadata (radius, intensity, tint).
  • onPreUpdate: NO
  • Data only
  • Read by GameScene
ResourceComponent
Marks entity as harvestable. Resource type + drop amount.
  • onPreUpdate: NO
  • resourceType: string
  • dropAmount: number
DecisionTreeComponent
Behavior tree nodes: Selector, Sequence, Condition, Action. Pre-built trees for melee/ranged/crawler.
  • Selector: first success
  • Sequence: all must pass
  • Melee/Ranged/Crawler
BonfireAnimComponent
Animated bonfire with breathing-idle frames + scale pulse.
  • onPreUpdate: YES
GlowEffectComponent
Pulsing glow overlay on bright pixels (eyes, runes).
  • onPreUpdate: YES
SpriteRendererComponent
Simple 8-dir sprite switching by velocity. Legacy (replaced by AnimatedSprite).
  • onPreUpdate: YES

Bot AI Decision Tree

Evaluated every frame. First matching leaf wins. Priority top-to-bottom.

SURVIVE & PROGRESS
SURVIVE
Critical HP Retreat (HP < 25%, unless 1 weak enemy left)
Dodge Projectile (evasion urgency > 1.5)
Surrounded (3+ enemies close, HP < 50%)
Defend Camp (enemy within 250px of bonfire)
Counter-Attack (projectile heading toward player → kill shooter)
Kite (surrounded by 2+ melee, step back + counter)
FIRE DYING (fuel < 35%)
Feed Fire (if has wood)
Chop for Fire (urgent)
COMBAT (enemy in sight range 350px)
Kill Enemy (stand and fight, dodge projectiles)
Feed Fire (maintenance, fuel < 85%)
Build (if can afford → walk to spot)
Gather for Build
Pick Up drops for build (nearest needed drop < 200px)
Chop/Mine for build (specific resource type)
Level Up Fire
Feed to level up
Gather wood for level up
Pick Up Nearby Drops (< 120px)
Gather Wood (for fire fuel)
Camp Idle (orbit bonfire slowly)

Enemy Decision Trees

Melee Tree
Attack (target < 30px) → stand and hit
Chase (target in sight 350px) → A* pathfind
Wander (random direction)
No flee — fights to the death
Ranged Tree
Flee (target < 50% attack range)
Orbit + Shoot (in attack range)
Chase (in sight range)
Wander
Crawler Tree
Chase bonfire (always)
Wander (fallback)
Targets bonfire, ignores player
